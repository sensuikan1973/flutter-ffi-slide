# Flutter FFI Slide
At [Flutter Meetup Tokyo #10](https://flutter-jp.connpass.com/event/134921/)  
See **https://speakerdeck.com/sensuikan1973/ffi-in-flutter**

## Development

### preview
```sh
yarn install
yarn start
```

### build HTML
```sh
yarn build
```

### presentation on local
```sh
yarn pdf
pdfpc slide.pdf
```
